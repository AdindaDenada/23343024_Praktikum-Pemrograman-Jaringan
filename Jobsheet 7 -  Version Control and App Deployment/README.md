# Jobsheet 7 – Manajemen Kontrol Versi & Diseminasi Aplikasi

Melalui praktikum ini, saya telah mendalami mekanisme pengelolaan kode sumber aplikasi Node.js menggunakan *Version Control System* (Git) serta prosedur **deployment** aplikasi ke lingkungan produksi. Materi ini memberikan pemahaman mendalam mengenai penggunaan Git dalam melacak riwayat perubahan kode, manajemen repositori di GitHub, tata kelola komit, serta konfigurasi file `.gitignore` untuk menyaring berkas yang tidak perlu dilacak oleh sistem.

Selain itu, modul ini membedah proses publikasi aplikasi web berbasis Express.js ke platform *cloud* **Vercel**. Fokus utama mencakup integrasi repositori GitHub dengan layanan cloud, sinkronisasi variabel lingkungan (*environment*), serta memastikan aplikasi dapat diakses secara publik melalui URL resmi.

# Kesimpulan

Berdasarkan hasil praktikum pada Jobsheet 7 ini, saya dapat menarik beberapa kesimpulan utama:

1. **Signifikansi Kontrol Versi:** Sistem kontrol versi seperti Git merupakan pilar krusial dalam pengembangan perangkat lunak modern. Git memungkinkan pengembang mengelola perubahan kode secara terstruktur dan kolaboratif, sementara GitHub berfungsi sebagai infrastruktur penyimpanan terpusat yang aman.
2. **Transisi Development ke Production:** Saya memahami alur pemindahan aplikasi Node.js dari lingkungan pengembangan lokal ke infrastruktur server awan. Proses ini menuntut ketelitian dalam menyesuaikan konfigurasi, seperti pengaturan *port* dan skrip aplikasi, agar sistem tetap stabil di lingkungan produksi.
3. **Workflow Backend yang Komprehensif:** Praktikum ini memberikan gambaran menyeluruh mengenai *workflow* pengembangan backend—mulai dari tahap penulisan kode (*development*), manajemen riwayat (*version control*), hingga tahap publikasi online (*deployment*).

Dengan menyelesaikan latihan ini, saya tidak hanya mampu membangun fungsionalitas aplikasi di komputer lokal, tetapi juga memiliki kompetensi untuk mempublikasikan karya tersebut agar dapat digunakan secara luas oleh pengguna melalui jaringan internet.
